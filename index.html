# 第一章 向量简介

线性代数的核心运算均是关于向量的。

对于两个向量 v⃗\vec{v}v 和 w⃗\vec{w}w ， 我们通过数乘向量 cv⃗c\vec{v}cv , dw⃗d\vec{w}dw 并且将其相加组合为 cv⃗+dw⃗c\vec{v}+d\vec{w}cv+dw 来进行线性组合的操作。

Liner Combination:cv⃗+dw⃗=c[11]+d[23]=[c+2dc+3d] Liner\ Combination:\quad c\vec{v} + d\vec{w} = c\begin{bmatrix} 1\\1 \end{bmatrix} + d\begin{bmatrix} 2\\3 \end{bmatrix} = \begin{bmatrix} c + 2d \\ c + 3d \\ \end{bmatrix} Liner Combination:cv+dw=c[11​]+d[23​]=[c+2dc+3d​]
Examplev⃗+w⃗=[11]+[23]=[34] is the combination with c=d=1 Example\quad \vec{v} + \vec{w} = \begin{bmatrix} 1\\1 \end{bmatrix} + \begin{bmatrix} 2\\3 \end{bmatrix} = \begin{bmatrix} 3\\4 \end{bmatrix} \ is\ the\ combination\ with\ c = d = 1 Examplev+w=[11​]+[23​]=[34​] is the combination with c=d=1

若 cv⃗c\vec{v}cv 与 dw⃗d\vec{w}dw 不共线， 则 v⃗\vec{v}v 和 w⃗\vec{w}w 所有的线性组合充满了整个二维空间。
以此类推到四维空间 u⃗\vec{u}u , v⃗\vec{v}v , w⃗\vec{w}w , z⃗\vec{z}z 四个不共面的向量的线性组合可充满整个四维空间。
由此，自然类推至n维空间，这就是线性代数优美的特性。

本章着眼于介绍向量及其基本运算和性质，引入，构建线性代数知识体系的基础。

# Java 基础

Java 是一个完全面向对象的语言，提供了一系列数据抽象和封装的工具方法，这一点和算法设计与应用的理念不谋而合。

我们将算法拆分成实现层和应用层两个层面，实现层专注于算法逻辑和思想，提供一层通用的 API(Application Programing Interface, 应用程序编程接口)，而应用层关注将具体问题进行算法建模，从而可以应用一些设计优秀的算法去解决实际问题。

而Java语言正式利用面向对象的思想进行模块化编程的，可以完美符合算法设计需求。

简而言之，Java 面向对象增强了模块化编程，将实现和用例完全拆分，这样的特性可以帮助我们更加清晰地进行算法设计和应用。
